# 2.Activity生命周期图

![image-20260327144233657](..\imgs\Activity_Life_Cycle.png)

- 首次启动Activity，执行了三个方法，分别是onCreate()、onStart()、onResume()
- 按Home键返回，执行了两个方法，分别是onPause()、onStop()
- 这时候再次点击应用程序，执行了三个方法，分别是onRestart()、onStart()、onResume()
- 如果把后台给杀死了，这时候就会执行onDestory()
- 如果启动Activity后，翻转了屏幕，这时候方法的执行顺序是onCreate()、onStart()、onResume()、onPause()、onStop()、onDestory()、onCreate()、onStart()、onResume()

```java

        Log.d(TAG, "onStop: ");
    }

        Log.d(TAG, "onRestart: ");
```

# 3.控件

在现代程序的框架体系里面，我们使用MVC架构，即**Model-View-Controller**，在这个模式里面，控件就是一个View。 

## 3.1 TextView

如果想要将xml文件里的控件和代码绑定，就要用findViewById这个方法，可以传的参数是R.id.xxxx，其中id就是xml中控件取的id。如果想要引入一个图片就是R.drawable，如果是视频就是R.raw.audio1，如果是布局，就是R.layout.layout1

```java
package com.example.helloworld;
```
